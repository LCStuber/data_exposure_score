# Dados e amostragem

## Fonte de dados
O estudo usa a rede social **Bluesky** por permitir coleta via API aberta (AtProto) e acesso a conteúdos públicos.

## Volume coletado
- Coleta massiva com armazenamento em banco orientado a documentos
- Universo com dezenas de milhões de postagens e centenas de milhares de usuários (ver paper)

## Amostragem estatística
Para viabilizar análise e validação, foi utilizada uma amostra representativa de usuários com:
- nível de confiança: 95%
- margem de erro: 1% na proporção

O tamanho calculado foi arredondado para facilitar operação (ver paper).
