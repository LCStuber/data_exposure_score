# Ética e LGPD

Este projeto trabalha com dados **manifestamente públicos** em redes sociais.

## Princípios adotados
- Minimização: coleta apenas do necessário para análise
- Agregação: foco em métricas e padrões, não em expor indivíduos
- Transparência: documentação do processo e das limitações
- Reprodutibilidade responsável: dependente de credenciais e respeitando termos de uso

## Limitações
A LGPD impacta diretamente o escopo:
- não utilização de bases de vazamentos
- escolha de plataforma com API pública para viabilizar o experimento de forma ética
